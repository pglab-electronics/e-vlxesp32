# E-VLXESP32

**E-VLXESP32** is an ESP32-based development board by **PG LAB Electronics** designed to control **VELUX®** motorized skylight windows using original **VELUX®** wall remotes.

## About

The **E-VLXESP32** is an electronics development board with an integrated **temperature and humidity sensor**.

It connects to battery-powered **VELUX® wall remote controls** (models **KLI311, KLI312, KLI313**) and allows you to control **VELUX® motorized skylight windows** over your **local Wi-Fi network**.

This repository contains **ESPHome YAML configuration files** used to build the firmware for the E-VLXESP32 board.

## Features

- Local Wi-Fi control of **VELUX® wall remotes**
- Compatible with **ESPHome**
- Local-only operation (no cloud required)
- Integrated **temperature and humidity sensor**

## Supported Devices

- E-VLXESP32
